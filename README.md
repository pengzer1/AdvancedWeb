## 프로젝트 소개

- 프로젝트 설명
  : 새로운 동네 친구들을 찾아서 멀리 가지 않아도 가볍게 만날 수 있게끔 도와주는 사이트입니다.

- 프로젝트 목적
  : 1. 새로운 친구를 사귈 수 있게 해줍니다. 2. 취미가 비슷한 사람들끼리 만나고 싶은 사람들을 위해 사용될 수 있습니다. 3. 오프라인으로 친구를 사귀기 어려워하는 사람들이 좀 더 편안하게 친구를 만들게 해줍니다.
